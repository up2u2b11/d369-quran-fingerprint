## Core Finding

```
Abjad:     Peak root = 9 (14.0%)  — concentrated in {3, 9}
Special-6: Peak root = 5 (13.1%)  — concentrated in {5, 9}
```

**Each system reveals a different distributional layer.**

Both systems show non-uniform distributions (p ≈ 0), but the shape of the non-uniformity differs. This is consistent with the idea that the two encoding systems are sensitive to different structural features of the text.
